UTILISATION :

L’installation du projet se fait via la commande :
npm install -- global

Pour exécuter les exemples tapez :
npm run build-examples

Afin de manipuler les exemples, il faut lancer un serveur qui se trouve au chemin suivant :
examples/shared
Exécuter la commande :
node server.js

Le serveur sera lancé sur le port 8080 (Si vous avez déjà un autre serveur exécuté sur le port
8080 , vous devez le fermer ou bien changer le port), pour changer le port, il faut se rendre dans les fichiers suivants et mettre votre numéro de port souhaité:

  • \examples\shared\index.js à ligne 11
  • \examples\shared\web-socket.js à la ligne 5
  • \examples\webrtc-browser-interactive\index.browser.js à la ligne 50

Enfin, tapez l’URL de l’exemple (http://localhost:num_port/examples/webrtc-browser-interactive/index.html),ouvrez plusieurs onglets et essayez de connecter les clients entre eux.
Vous pouvez regarder la console pour savoir le fonctionnement de kad-webrtc.
